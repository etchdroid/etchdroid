EtchDroid — це програма з відкритим кодом, яка допомагає записувати образи на USB-накопичувачі.
Використовуйте її, щоб створити зав*антажувальний USB-носій із операційною системою, коли ваш ноутбук перестав працювати.

---

⭐️ Підтримувані пристрої ⭐️

✅ USB-флешки
✅ Адаптери для SD-карт через USB
❌ Зовнішні жорсткі диски / SSD через USB
❌ USB-концентратори та док-станції
❌ Вбудовані слоти для SD-карт
❌ Оптичні або дисководні накопичувачі
❌ Пристрої виключно з інтерфейсом Thunderbolt

---

⭐️ Підтримувані типи образів дисків ⭐️

✅ Сучасні образи GNU/Linux (включаючи Arch Linux, Ubuntu, Debian, Fedora, pop!_OS, Linux Mint, FreeBSD, BlissOS та багато інших)
✅ Образи для SD-карт Raspberry Pi (попередньо розархівуйте їх!)
❌ Офіційні ISO-образи Microsoft Windows
⚠️ Збірки Windows від спільноти, створені для EtchDroid (обережно: можуть містити віруси!)
❌ Дискові образи Apple DMG
❌ Застарілі образи GNU/Linux до 2010 року (наприклад, Damn Small Linux)
