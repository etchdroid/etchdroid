EtchDroid คือแอปพลิเคชันโอเพนซอร์สที่ช่วยให้คุณเขียนอิมเมจลงในไดรฟ์ USB

ใช้เพื่อสร้างไดรฟ์ USB ระบบปฏิบัติการที่สามารถบูตได้เมื่อแล็ปท็อปของคุณเสีย

⭐️ อุปกรณ์ที่รองรับ ⭐️

✅ แฟลชไดรฟ์ USB
✅ อะแดปเตอร์การ์ด SD USB
❌ ฮาร์ดไดรฟ์ USB / SSD
❌ ด็อกและฮับ USB
❌ ช่องใส่การ์ด SD ภายใน
❌ ไดรฟ์ออปติคัลหรือฟล็อปปี้ดิสก์
❌ อุปกรณ์ที่ใช้ Thunderbolt เท่านั้น


⭐️ ประเภทดิสก์อิมเมจที่รองรับ ⭐️

✅ อิมเมจระบบปฏิบัติการ GNU/Linux รุ่นใหม่ ได้แก่ Arch Linux, Ubuntu, Debian, Fedora, pop!_OS, Linux Mint, FreeBSD, BlissOS และอื่นๆ อีกมากมาย
✅ อิมเมจการ์ด SD Raspberry PI (ต้องแตกไฟล์ก่อน!)
❌ ISO อย่างเป็นทางการของ Microsoft Windows
⚠️ อิมเมจ Windows ที่สร้างโดยชุมชน สร้างขึ้นสำหรับ EtchDroid (โปรดระวัง: อาจมีไวรัส!)
❌ อิมเมจดิสก์ Apple DMG
❌ อิมเมจระบบปฏิบัติการ GNU/Linux รุ่นเก่ากว่า < 2010 เช่น Damn Small Linux

โค้ดอยู่บน GitHub: https://github.com/EtchDroid/EtchDroid
